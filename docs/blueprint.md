# **App Name**: SevaSetu

## Core Features:

- Symptom Input: Allows patients to input symptoms via text or speech.
- Multilingual AI Chat: The AI chatbot engages with patients in their local language, asks follow-up questions, and suggests possible diseases.
- Language Settings: A settings panel for users to easily switch between available languages.
- Automated Prescription Generation: Generates a prescription in the patient's language based on the AI's assessment.
- Prescription Delivery: Provides options to download the prescription as a PDF, send via WhatsApp (PDF link), or SMS in Unicode text.
- Speech-to-Text & Text-to-Speech: Utilizes Speech-to-Text to convert voice input to text, and Text-to-Speech to provide voice responses in the selected language. The tool must incorporate language when generating speech.
- Patient history tool: Maintains logs of conversations and prescriptions. The LLM powered tool uses the chat history to formulate suggestions to the user

## Style Guidelines:

- Primary color: Forest green (#388E3C) to evoke a sense of health, nature, and reliability.
- Background color: Light green (#E8F5E9), a very low saturation variation on forest green.
- Accent color: Muted orange (#E65100), which provides a contrast to the green, drawing attention to calls to action.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif, suitable for both headlines and body text.
- Use simple, clear icons related to healthcare and communication. Consider using open-source icon libraries like Font Awesome or Material Icons.
- Mobile-first design with a chatbot-style interface, large buttons, and a clear, readable font. Ensure accessibility with sufficient contrast and clear labels.
- Subtle animations for loading, transitions, and user feedback to improve user experience.
